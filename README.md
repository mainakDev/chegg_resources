## ‚úÖ Step 2: Install Python

## ‚úÖ Step 3: Install Jupyter Extension in VS Code
1. Open **VS Code**.
2. Go to **Extensions** (left sidebar).
3. Search for **"Jupyter"** and click **Install**.
4. Also install **Python extension** if prompted.

## üîç Troubleshooting
### 1. Kernel Not Found
- Ensure Python is installed and added to PATH.
- Restart VS Code after installing Python.

### 2. Missing Jupyter Extension
- Go to VS Code Extensions and install **Jupyter**.

### 3. Notebook Cells Not Running
- Check if the correct Python interpreter is selected in VS Code.
